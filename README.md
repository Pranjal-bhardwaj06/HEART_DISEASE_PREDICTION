# HEART_DISEASE_PREDICTION
This project is a Heart Disease Prediction System developed using Machine Learning and deployed using Streamlit.
It predicts whether a person is likely to have heart disease based on various medical parameters provided by the user.

The application provides a simple, interactive web interface for real-time prediction.

##  Features

Interactive web application using Streamlit

Real-time heart disease prediction

Machine Learning based classification

User-friendly input form

Fast and accurate results

## Input Parameters

The prediction is based on the following health parameters:

Age

Gender

Chest pain type

Resting blood pressure

Cholesterol level

Fasting blood sugar

Resting ECG results

Maximum heart rate achieved

Exercise induced angina
